#映画感想投稿サイト
##概要
この映画感想投稿サイトは、技術ブログを参考にしながら初めて自作したWebアプリケーションです。学習の一環として作成したため、実装中に直面した疑問点や不明点については、コード内にコメントとして記載しています。

参考にした技術ブログはこちらです: Django 初心者が簡単なアプリをつくる1 #Python - Qiita

##機能
- 監督の追加: 新しい映画監督の情報を追加できます。
- 映画タイトルの閲覧と視聴日時の記録: 映画のタイトルと視聴した日時を記録・閲覧できます。
- 感想（ログ）の投稿: 各映画に対する感想を投稿できます。
- 投稿内容の閲覧と削除: 投稿した感想を閲覧し、不要なものは削除できます。
- 感想の編集: 投稿済みの感想を後から編集できます。

##使用技術
バックエンド: Django
フロントエンド: Tailwind CSS（chatgpt）

##注意点
本プロジェクトは学習目的で作成されたものであり、コード内に学習中のコメントやメモが含まれています。実運用を目的としたものではありませんので、ご留意ください。
